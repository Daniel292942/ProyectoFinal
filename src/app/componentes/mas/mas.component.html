Imagenes



<img>

Agrega imagenes, especificando la propidad src como la ruta donde se encuentra guardada

Ejemplo

<imgsrc ="img/casa.jpeg">

Etiquetas de posicion



<section>

Representa una sección genérica de un documento. Sirve para determinar qué contenido corresponde a qué parte de un esquema.

<div>

Se utiliza para crear secciones grupales

<aside>

representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento

<footer>

representa una el pie de pagina
